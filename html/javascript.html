  • parseFloat()

    • Quand l'utiliser?

    Utilise le lorsque tu as un texte qui contient une chiffre, et que tu veux extraire ce nombre pour l’utiliser dans des calculs.
    Par exemple, si un utilisateur entre "123,45 euros", tu peux récupérer juste le nombre grâce à parseFloat().

    Exemples pour mieux comprendre :
    let prix = "250.75 euros";
    let nombre = parseFloat(prix);
    console.log(nombre); // Affiche 250.75

    Remarque sur le point décimal :
    parseFloat() utilise uniquement le point comme séparateur décimal (et non la virgule). parseFloat("12,34"); // NaN
    parseFloat("12.34"); // 12.34

    Texte sans nombre au début donne NaN (Not a Number) :
    let nombre = parseFloat("abc123");
    console.log(nombre); // NaN
